<h3>实用小工具与常见问题解答 (FAQ)</h3>
<p>掌握一些基础的诊断工具和方法，能帮助您更快速地定位问题根源。</p>
<ul>
<li>
<p><strong>1. 为什么我的订阅链接无法更新？</strong></p>
<p>这通常由几个原因导致：首先，请检查您的本地网络是否能正常访问该链接地址。其次，链接本身可能已过期或被服务商更换。最后，部分地区的网络环境可能干扰了对订阅服务器的访问。您可以尝试更换网络环境（如切换 Wi-Fi 或蜂窝数据）后再次尝试更新。</p>
</li>
<li>
<p><strong>2. 如何判断是本地网络问题还是节点问题？</strong></p>
<p>您可以使用系统自带的 `ping` 或 `traceroute` 命令进行简单诊断。例如，在无法访问某个网站时，先关闭代理，然后打开终端或命令提示符，执行：</p>
<p><code>ping google.com</code></p>
<p>如果本地网络正常，但开启代理后依然无法访问，则大概率是 <strong>Clash 节点</strong> 或规则配置存在问题。反之，如果关闭代理后 `ping` 也不通，则说明是您的本地网络连接故障。</p>
</li>
<li>
<p><strong>3. Clash 提示 "YAML" 格式错误怎么办？</strong></p>
<p>Clash 的配置文件基于 YAML 语言，对缩进和语法要求非常严格。手动修改配置文件时，一个多余的空格都可能导致解析失败。建议使用专业的文本编辑器（如 VS Code 机场加速并安装 YAML 插件）来检查语法，或使用在线的 YAML 验证工具来定位错误行。</p>
</li>
<li>
<p><strong>4. 为什么开启代理后，国内网站访问变慢了？</strong></p>
<p>这是典型的分流规则配置不当。请确保您的代理模式设置机场购买为 “Rule” (规则) 模式，并且规则列表中包含了对国内常用域名的 “DIRECT” (直连) 策略。这样可以确保国内流量不走代理，从而保证访问速度。</p>
</li>
</ul>
